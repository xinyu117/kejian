## 🎯 主要功能说明

### 用户权限
- **普通用户**: 只能查看免费课件（课件1-10）
- **VIP用户**: 可以查看所有课件（课件1-20）

### 支付功能
- 点击付费课件会弹出VIP升级提示
- 支持微信扫码支付（模拟）
- 可以使用"模拟支付成功"按钮快速测试

### 课件管理
- 所有课件文件存储在 `coursewares/` 目录
- 课件使用Git进行版本管理
- 支持HTML格式的课件上传

## 🚀 开发模式启动

如果需要重新启动应用：

```bash
# 停止当前应用 (Ctrl+C)
# 然后运行：
npm run dev
```

## 📁 项目结构

```
App/
├── src/                    # 后端源码
├── views/                  # EJS模板文件
├── public/                 # 静态资源
├── coursewares/            # 课件文件(Git管理)
├── scripts/                # 工具脚本
└── data.db                 # SQLite数据库
```
