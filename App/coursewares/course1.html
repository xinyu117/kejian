数学基础教程

分类: 数学 | 难度: 基础

第一章：数的认识

数学是研究数量、结构、变化以及空间等概念的一门学科。在这一章中，我们将学习数的基本概念。

1.1 自然数

自然数是用来计数的数，包括1, 2, 3, 4, 5, ...

例题 1.1

小明有3个苹果，小红给了他2个苹果，现在小明一共有多少个苹果？

解答: 3 + 2 = 5，所以小明一共有5个苹果。

1.2 整数

整数包括正整数、负整数和零。例如：..., -2, -1, 0, 1, 2, ...

第二章：四则运算

2.1 加法运算

加法是数学中最基本的运算之一。加法的符号是"+"。

例题 2.1

计算：15 + 28 = ?

解答:

  15
+ 28
----
  43

2.2 减法运算

减法是加法的逆运算。减法的符号是"-"。

例题 2.2

计算：50 - 23 = ?

解答: 50 - 23 = 27

第三章：应用练习

练习题

  1. 计算：123 + 456 = ?
  2. 计算：789 - 234 = ?
  3. 小李买了一支笔花了5元，买了一本书花了12元，他一共花了多少钱？
  4. 教室里原来有30个学生，走了8个学生，现在还有多少个学生？

练习答案

  1. 123 + 456 = 579
  2. 789 - 234 = 555
  3. 5 + 12 = 17元
  4. 30 - 8 = 22个学生
